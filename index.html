My Work

GetInked

GetInked is a tattoo app which allows users to create their own profile, get connected with other people passionate about tattoos and be able to book in for consultations and appointments with tattoo artists. This is my very first project which I created for the UX Immersion CareerFoundry course.

Role: User Research, UX Designer, Interaction Design
Tools: Balsamiq, Adobe XD, Optimal Workshop, Usability Hub
Duration: 4 Months
